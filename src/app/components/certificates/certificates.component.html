MINHAS CERTIFICAÇÕES

JavaScript - B7Web

Carga Horária: 77 Horas

O curso de JavaScript da B7Web oferece uma abordagem abrangente para aprender JavaScript. Os tópicos incluem: introdução à linguagem e sintaxe, variáveis e tipos de dados, estruturas de controle, funções, manipulação do DOM, eventos, Ajax e solicitações HTTP, manipulação de formulários e tratamento de exceções. Os alunos também exploram programação orientada a objetos, recursos avançados do ECMAScript e muito mais. Este curso é projetado para fornecer uma base sólida em JavaScript, abrangendo desde conceitos básicos até recursos avançados, permitindo que os alunos desenvolvam habilidades sólidas de programação web.

TypeScript - B7Web

Carga Horária: 20 Horas

O curso de TypeScript da B7Web é uma excelente opção para aprender essa linguagem de programação fortemente tipada. Os tópicos incluem: introdução ao TypeScript e sua relação com JavaScript, declaração de tipos de dados, interfaces, classes e herança, uso de módulos, configuração de ambiente de desenvolvimento, manipulação do DOM, integração com bibliotecas JavaScript e muito mais. Com foco na criação de aplicativos web modernos, este curso oferece uma base sólida para desenvolvedores que desejam utilizar TypeScript para criar aplicativos mais robustos e escaláveis.

Java POO - Alura

Carga Horária: 8 Horas

O curso de Programação Orientada a Objetos (POO) da Alura em Java abrange uma ampla variedade de tópicos essenciais, incluindo: conceitos fundamentais de Java e POO, coleções, tratamento de exceções, programação funcional, persistência de dados com JDBC, frameworks como Spring e Hibernate, testes unitários, padrões de projeto, desenvolvimento web com Servlets e JSP, criação de APIs REST, e culmina em um projeto prático abrangente. Este curso fornece uma base sólida para desenvolvedores que desejam dominar Java e suas aplicações em desenvolvimento de software e web.

Java polimorfismo: Herança & Interfaces - Alura

Carga Horária: 16 Horas

O curso de Java Polimorfismo, Herança e Interfaces da Alura abrange os princípios essenciais de orientação a objetos em Java. Os tópicos incluem: polimorfismo, herança, interfaces, métodos abstratos, classes abstratas, sobrescrita e sobrecarga, uso de referências polimórficas, casting e aplicação prática por meio de exercícios. Este curso oferece aos desenvolvedores uma base sólida para aproveitar ao máximo a flexibilidade e a reutilização de código proporcionadas pelo Java, com foco em polimorfismo, herança e interfaces.

Java: TDD & Testes Unitários - Alura

Carga Horária: 8 Horas

O curso de Java: TDD & Testes Unitários da Alura abrange tópicos essenciais relacionados ao desenvolvimento orientado por testes (TDD) e testes unitários em Java. Os principais temas incluem: conceitos de TDD, criação de testes unitários com JUnit, estratégias de escrita de testes, refatoração, testes de integração, testes parametrizados, e aplicação prática por meio de projetos. Este curso capacita desenvolvedores a adotar práticas de TDD para garantir a qualidade do código Java e criar testes unitários eficazes para seus projetos.

Angular: Web Components - DIO

Carga Horária: 3 Horas

O curso "Angular: Web Components" da DIO aborda uma ampla gama de tópicos essenciais, incluindo: introdução ao Angular, criação de componentes e templates, uso de diretivas, organização em módulos, roteamento, injeção de dependência, manipulação de formulários, integração com serviços web, teste unitário e desenvolvimento de componentes web reutilizáveis com Web Components. Este curso é projetado para fornecer aos alunos habilidades abrangentes em Angular, permitindo-lhes criar aplicativos web robustos e componentes web personalizados para uma experiência de desenvolvimento web completa.

Angular: Módulos - DIO

Carga Horária: 1 Horas

O curso "Angular: Módulos" abrange tópicos essenciais relacionados ao uso de módulos em Angular. Os tópicos incluem: introdução ao Angular e sua arquitetura de módulos, criação e organização de módulos, compartilhamento de funcionalidades, módulos de funcionalidade, carregamento preguiçoso, escopo de módulo, integração de módulos externos e melhores práticas de uso de módulos. Este curso capacita desenvolvedores a estruturar e organizar seus aplicativos Angular de maneira eficaz, melhorando a manutenção e escalabilidade dos projetos.
